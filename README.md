## Install

`go get -u github.com/sgaynetdinov/go-yandex-disk`


## Step 0

```
import (
    yandexdisk "github.com/sgaynetdinov/go-yandex-disk"
)

func main() {
    client := yandexdisk.NewClient("YOUR_TOKEN")
}
```
